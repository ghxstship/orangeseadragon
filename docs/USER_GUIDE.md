## PROJECT MANAGEMENT

### Creating a New Project
1. Navigate to **Projects** → **Productions**
2. Click **"Add Project"** button
3. Fill in basic information:
   - **Project Name** - Clear, descriptive title
   - **Project Type** - Film, TV, Event, etc.
   - **Client** - Select from company directory
   - **Project Manager** - Assign team lead
4. Set timeline:
   - **Start Date** - When work begins
   - **End Date** - Target completion date
   - **Budget** - Total project budget
5. Add team members and assign roles
6. Click **"Create Project"**

### Project Workflow
Projects follow a structured lifecycle:

1. **Planning** - Initial setup and resource allocation
2. **Active** - Day-to-day production work
3. **On Hold** - Temporary pause (e.g., waiting for approvals)
4. **Completed** - Final deliverables delivered
5. **Archived** - Historical record keeping

### Managing Project Tasks
Each project contains multiple tasks:

- **Create tasks** from the project detail page
- **Assign team members** to specific tasks
- **Set due dates** and priority levels
- **Track progress** through status updates
- **Add dependencies** between related tasks

### Project Budgeting
Track financial aspects:

- **Set initial budget** when creating project
- **Monitor expenses** as they occur
- **Track time** against billable hours
- **Generate reports** on budget vs. actual spending

## ASSET MANAGEMENT

### Adding Equipment
1. Go to **Assets** section
2. Click **"Add Asset"**
3. Enter asset details:
   - **Asset Tag** - Unique identifier
   - **Name** - Equipment name
   - **Category** - Camera, Lighting, Audio, etc.
   - **Purchase Date** - When acquired
   - **Current Value** - Depreciated value
4. Assign to location and set status

### Asset Tracking
- **Check out/in** equipment for projects
- **Schedule maintenance** and repairs
- **Track depreciation** automatically
- **Generate reports** on asset utilization
- **QR codes** for quick scanning and identification

### Inventory Management
- **Stock levels** for consumable items
- **Reorder alerts** when supplies run low
- **Supplier tracking** for procurement
- **Cost tracking** per item/category
